This website is a training project. It will begin as avery simple website and progressively increase in design and complexity. This website is programed in python programming language and uses the Flask web development microframework.

The project will evolve as follows:

  • I will begin by creating html templates rendered by my decorated functions in the source code
  • I will then add some css templates
  • I will then add some Javascript
  • I will then pause my progress on the front-end to focus on the back end
  • On the back end, I will:
    • Build a web-scraper that collect some data and connect it to the web app
    • Create a database, both SQL and NoSQL, that stores this data
    • create user log ins and passwords
    • add some user interactivity
    • Let the user post data of various forms
    • create an interactive data visualization feature
    • create a GUI
    • Find some way to practice git and github in the process
    • Find some way to practice Gulp in the process
    • Explore various Javascript technologies such as Node.js or React
    • Explore functions such as time and datetime in practice
    • Explore blockchain applications
    • Practice testing
    • Practice maintenance
    • Practice debugging
    • Practice UX experiments
    • Deploy the site to an internet server (Heroku or AWS)

That's it for now, but don't worry, this project will grow very fast
